Session Guide

Step-by-Step Guide

Teacher flow · Student flow
Teacher — Mission Setup
1
Open the Create tab

Tap Create in the navigation. You'll see the Mission Generator.

2
Pick a domain & level

Select one of 7 domains (e.g. Drawing). Then pick the difficulty level that fits your class today.

Foundation → Expert
3
Hit Generate

The AI writes a unique mission using only the materials marked available for your class. Takes 5–10 seconds.

4
Review & share

Read the mission aloud, or copy it and paste into a screen or chat. Tap Regenerate if it doesn't fit.

Copy Mission button
5
Monitor in Class tab

Switch to the Class tab during the session. See every student's XP, level, and mission count — live.

6
Send parent reports

After class, go to the Report tab. Generate a shareable link and send it to parents by email or WhatsApp. No login required to view.

Student — Session Flow
1
Sign up or log in

Go to art-os-nu.vercel.app. Tap I'm a Student. Enter name, email, password, and the join code DICE2026.

First time only
2
Get their mission

Tap Mission in the nav. Pick a domain and difficulty, then tap Generate to receive a unique challenge.

3
Create the artwork

Work on the challenge with physical materials. Use the hint system (💡) if stuck — each student starts with 5 hint credits.

5 hints available
4
Earn XP

Completing a mission awards XP. Students level up every 200 XP — Foundation → Beginner → Intermediate → Advanced → Expert.

5
Add to portfolio

Tap Portfolio to add the finished piece. Toggle it public to get a shareable link — anyone can view it, no login needed.

6
Join a clan

Tap Clans to join a creative tribe, react to others' work, and take on clan challenges set by the teacher.

First login triggers a guided tour — the platform walks new users through each feature automatically. Students don't need to read anything.

Advanced Features & Troubleshooting

Reference Sheet

Clan challenges · Lesson library · Fixes
Running a Clan Challenge (Optional — Great for End of Session)

① Open Clans

Tap Clans in the nav. Find a clan your students are in (or create one). Tap into it.

② Create Challenge

Tap ⚡ New Challenge. Set a title, domain, deadline, and XP reward. Students see it immediately.

③ Submissions Come In

Students tap the challenge card and submit a description of their work. Each earns the XP reward.

Using the Lesson Library (For Curriculum Support)

50 structured lessons

Tap Lessons in the nav. Browse 50 lessons across 7 domains, Levels 1–50. Each has learning objectives, a detailed activity, materials list, and 3 reflection prompts.

AI Improve button

Each lesson has an AI Improve button that rewrites the activity description to better fit your class context. Useful for adapting a lesson on the fly.

Troubleshooting
Problem Fix
Student can't sign up — join code rejected Check they're typing exactly DICE2026 (all caps). If still failing, use the demo student account for that session.
Mission generation shows an error Usually a network blip. Wait 10 seconds and tap Generate again. If it persists, check your internet connection.
Mission uses materials we don't have Go to Create → Class Materials and toggle off the unavailable items. Regenerate the mission — it will only suggest available materials.
Student forgot their password Use the Forgot password link on the sign-in page. They'll get a reset email. As a backup, share the demo student login.
XP not showing after completing a mission Refresh the page. XP is awarded server-side; it may take a few seconds to appear in the student's profile.
Class tab shows no students Students need to have logged in at least once for them to appear. Check they're registered and signed in.
AI Improve in lessons isn't responding The API key may not be set on the server. Contact the PrimeLuck team — this is a backend configuration, not a user fix.
Page looks broken on an older phone Ask the student to open in Chrome or Safari (latest version). Avoid Samsung Internet or old browsers.
Something not listed here? Screenshot the error and WhatsApp it to the PrimeLuck team immediately. We monitor during pilot sessions and can fix most things remotely within minutes.
