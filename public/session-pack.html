Teacher — Mission Setup
1
Log in as Teacher Create tab

Sign in at art-os-nu.vercel.app. You land on the Class view. Tap Create in the nav.

2
Check materials first Create → Materials

Tap the Class Materials tab. Toggle on only what's physically available today. Missions will suggest only available materials.

3
Pick domain + level Create → Generator

Tap Mission Generator. Select a domain (Drawing, Painting, Colour, etc.) and difficulty level for your class. The XP reward is shown per level.

4
Generate — preview — decide

Tap Generate Mission. A unique creative brief appears on the right. Read it — hit Regenerate if it doesn't fit. When happy, tap Copy Mission to share with students verbally or paste it.

5
Watch your class Class tab

Tap Class in the nav during the session. See every student, their XP, rank, and missions completed — live. Auto-refreshes every 30 seconds.

6
End of session — send the report Report tab

Tap Report in the nav. Tap Generate shareable link. Copy and WhatsApp/email it to parents — no login required on their end.

7
Curriculum support Lessons tab

50 structured lessons across 7 domains. Each has objectives, activity, materials, and 3 reflection prompts. Tap AI Improve to rewrite any activity for your class context.

Student — Session Flow
1
Sign up or log in

Go to art-os-nu.vercel.app. Tap I'm a Student. Enter your name, email, password, and the join code DICE2026. Takes under 60 seconds.

2
Complete the tour first login only

A 4-step guided tour appears. Tap through it — it shows exactly where each feature lives. It only shows once and takes 30 seconds.

3
Get a mission Mission tab

Tap Mission in the nav. Pick a domain photo, then a level. Tap Generate Mission. A unique creative brief appears — read it, then make art.

4
Use hints if stuck inside the mission

Below the mission card are 3 locked hint slots. Each costs 1 hint credit (you start with 5). Tap a slot to reveal a technique clue with a visual reference. Use wisely.

5
Complete and earn XP

When you finish the artwork, tap Mark Complete on the mission card. XP is awarded automatically — watch your level ring fill up.

6
Add work to portfolio Portfolio tab

Tap Portfolio. Tap + Add Work. Give it a title and reflection. Toggle it public to get a shareable link — anyone can see it without logging in.

7
Join a clan + take challenges Clans tab

Tap Clans. Find a clan and join it. React to others' work, post updates, and take on clan challenges set by the teacher to earn bonus XP.

Running a Clan Challenge — Optional, Great for Final 10 Minutes

① Open a Clan

Tap Clans in the nav. Find a clan your students are in (or create one first). Tap into it to open the clan view.

② Create Challenge

Tap ⚡ New Challenge. Set a title, domain, deadline, and XP reward. Students see it immediately on their Clans tab — no refresh needed.

③ Submissions & XP

Students tap the challenge card and submit a description of their response. Each submission earns the full XP reward you set.

Sending the Parent Report — End of Session

📬 Weekly student summary — one tap to share

At the end of the session, tap Report in the nav. The page shows a summary of the student's week: XP earned, missions completed, portfolio count, domains explored, and achievements unlocked. Tap Generate shareable link — the link copies to your clipboard automatically. Send it to parents via WhatsApp, email, or the school's messaging platform.

Parents don't need an account — the link opens a public summary page. It auto-updates each week. The teacher can generate one report link per student and re-send the same link each week.

Troubleshooting
ProblemFix
Student join code rejected Must be exactly DICE2026 (all caps, no spaces). If it still fails, use the demo student account: student@primeluckart.com / PrimeluckArtOs
Mission uses materials we don't have Go to Create → Class Materials and toggle off the unavailable items. Regenerate — it will only suggest what's available.
Mission generation shows an error Usually a network blip. Wait 10 seconds and tap Generate again. Check internet connection. If it persists, write the mission brief manually or use the Lessons tab.
Student forgot their password Use the Forgot password link on the sign-in screen — a reset email arrives within 2 minutes. As backup, share the demo student login.
XP not showing after completing a mission Pull to refresh or reload the page. XP is awarded server-side and may take a few seconds to appear in the student's level ring.
Class tab shows no students Students must sign up and log in first. They appear automatically once registered. Confirm they're on the right URL and used join code DICE2026.
Parent report link not working Tap Report in the nav, then Generate shareable link again — the link is public and doesn't expire. Check you copied the full URL.
AI Improve in Lessons isn't responding The API key needs to be set in the server config. Contact the PrimeLuck team — this is a backend configuration, not a user fix.
Page looks broken on a student's phone Ask them to open in Chrome or Safari (latest). Samsung Internet and older browsers may have display issues. The platform is designed for mobile — mobile Chrome works best.
Something not listed here? Screenshot the error and contact the PrimeLuck team immediately. Most issues can be resolved remotely within minutes during a pilot session.
